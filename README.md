#  Análise de Aporte de Ações (DY Analyzer B3)
# Autor: Mailon Agusto



Projeto Spring Boot desenvolvido para realizar a análise fundamentalista de ações listadas na B3 (Bolsa de Valores do Brasil) com base no **Dividend Yield (DY)**, comparando o rendimento projetado com um critério de aporte mínimo definido pelo usuário.

A aplicação possui uma interface web (front-end) simples construída com **Thymeleaf**, facilitando a interação e visualização dos resultados.

---

##  Tecnologias Utilizadas

* **Backend:** Java 17+
* **Framework:** Spring Boot 3.x
* **API de Dados:** Alpha Vantage (para preços e dividendos)
* **Web:** Spring Web (Controller)
* **Templating:** Thymeleaf (para renderização HTML)
* **Comunicação API:** Spring WebFlux (WebClient)
* **Gerenciador de Dependências:** Maven
